# theGist

A transcript ingestion and knowledge extraction pipeline that summarizes 
video content, builds a searchable knowledge base, and provides tools 
to actively learn and test the extracted insights.

## Features (In Progress)
- Transcript ingestion via yt-dlp
- Audio transcription via Whisper
- Insight extraction via local LLM (Ollama)
- Semantic search via ChromaDB
- Knowledge testing and trivia layer

## Tech Stack
- Python 3.11
- yt-dlp
- OpenAI Whisper
- Ollama (Llama 3)
- ChromaDB
- Sentence Transformers
- Streamlit
